# Ironbark Training

**Movement. Longevity. Follow-through.**

Website for Dave Conway's premium health and longevity coaching practice for men over 40.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deploy to Vercel

Push to GitHub, connect to Vercel, deploy.

## Customisation

- Contact details: `src/components/Contact.tsx`
- Testimonials: `src/components/Testimonials.tsx`
- Pricing: `src/components/Services.tsx`
